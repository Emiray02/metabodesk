Quick Start

5 minutes

Follow these steps to get Metabodesk running locally and connect your first workspace.

  1. Install the Metabodesk desktop app.
  2. Create a workspace and generate an API key.
  3. Configure your first data source.
curl -X POST https://api.metabodesk.com/v1/workspaces \
  -H "Authorization: Bearer <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"name":"Demo Workspace"}'
Tip: Use environment variables to store API keys.

API Reference

REST

List Workspaces

GET /v1/workspaces

GET https://api.metabodesk.com/v1/workspaces
Authorization: Bearer <API_KEY>
Field Type Description
id string Workspace identifier
name string Human readable name
createdAt datetime ISO-8601 timestamp
Note: Rate limits default to 100 requests/minute.
